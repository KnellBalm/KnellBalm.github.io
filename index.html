데이터 시각화를 통한 플랫폼 활성화 전략

Problem

수원시는 방대한 데이터를 보유했지만, 시민들이 체감할 수 있는 서비스가 부족하여 플랫폼 활용도가 저조했습니다. 데이터를 단순 공개하는 것을 넘어, 시민의 삶에 직접적인 가치를 제공하는 '킬러 콘텐츠' 개발이 필요했습니다.

Action (My Solution Process)

  1. 시민 중심 서비스 기획: '내 주변 착한가격업소' 등 시민들이 일상에서 궁금해할 만한 주제를 발굴하고, 10여 종의 대민 서비스 대시보드를 직접 기획했습니다.
  2. 데이터 융합 및 분석: 공공 데이터와 민간 데이터를 융합하여 '수원시 산업구조' 등 심층 분석을 수행, 정책 부서에 데이터 기반의 SWOT 분석 자료를 제공했습니다.
  3. 사용자 친화적 시각화 개발: BI 툴과 GIS 엔진을 활용하여 복잡한 데이터를 누구나 쉽게 이해할 수 있는 인터랙티브 대시보드로 구현했습니다.

Impact (My Contribution & Result)

  • 플랫폼 대표 서비스 성장: 제가 개발을 주도한 서비스가 누적 조회수 3.1만 회를 돌파하며 플랫폼에서 가장 성공적인 서비스로 자리매김했습니다.
  • 데이터의 공공 가치 실현: 타 지자체에서 벤치마킹하는 우수 사례가 되었고 데이터 기반 행정의 가치를 증명했습니다.

Learnings (배운 점)

사용자 데이터(페이지뷰, 체류시간 등)를 직접 분석하고 피드백을 반영하며 서비스를 개선하는 과정을 통해, 데이터 제품의 전체 수명 주기를 경험하고 사용자 중심의 사고방식을 체득할 수 있었습니다.

머신러닝 기반 교통사고 위험 예측

Problem

교통사고, 무인단속장비 데이터가 분산되어 있어 종합적인 분석과 선제적인 안전 대책 수립에 어려움이 있었습니다. 특히, 관련 민원 응대에 많은 시간이 소요되어 내부 업무 효율 저하가 문제였습니다.

Action (My Solution Process)

  1. 교통사고 위험지역 예측 모델(ML) 개발: 사고 이력과 도로 형태 등 공간 정보를 `PostGIS`로 융합하여, 미래 사고 위험도를 예측하는 머신러닝 모델을 개발했습니다.
  2. 업무 지원 대시보드 개발: 분산된 데이터 소스를 통합하고 분석 결과를 시각화하여, 교통 관련 민원 응대에 필요한 모든 정보를 한 화면에서 확인할 수 있는 대시보드를 기획, 개발했습니다.

Impact (My Contribution & Result)

  • 민원 응대 시간 80% 단축: 개발된 대시보드를 통해 관련 데이터 조회 시간이 평균 5분에서 1분으로 극적으로 감소하여, 담당자의 업무 효율성을 크게 향상시켰습니다.
  • 선제적 사고 예방 체계 지원: 사고 위험 예측 모델 결과는 경찰청에 제공되어 순찰 강화 및 교통 안전 시설물 추가 설치 등 선제적 예방 활동에 활용될 예정입니다.

Learnings (배운 점)

현업 부서와의 긴밀한 인터뷰를 통해 실질적인 문제(Pain Point)를 정의하는 것이 프로젝트 성공의 핵심임을 깨달았습니다. 기술적인 성과뿐만 아니라, 실제 사용자의 업무 프로세스를 개선했을 때 더 큰 만족을 느낄 수 있었습니다.

전기차 충전 인프라 수요 예측

Problem

전기차 보급이 확산됨에 따라, 한정된 예산으로 최적의 장소에 충전소를 신설해야 하는 정책적 과제가 대두되었습니다. 경험에 의존하는 방식은 비효율적 투자로 이어질 수 있어, 데이터 기반의 미래 수요 예측이 시급했습니다.

Action (My Solution Process)

  1. 다차원적 이용 패턴 분석: 충전기 로그 데이터를 분석하여, '충전 유형(급속/완속)'과 '설치 장소(아파트, 휴게소 등)'가 이용률에 미치는 핵심적인 차이를 발견하고 모델의 주요 피처로 활용했습니다.
  2. 시계열 예측 모델 개발: 페이스북의 `Prophet` 라이브러리를 활용하여, 개별 충전소의 고유 패턴(주간/계절별 주기성 등)을 모두 반영하는 예측 모델을 개발했습니다.

Impact (My Contribution & Result)

  • 데이터 기반 정책 수립의 기틀 마련: 저의 예측 모델은 향후 충전 인프라 확충 사업에서 최우선 순위 지역을 선정하는 핵심적인 근거 자료로 채택되었습니다.
  • 객관적 의사결정 지원: 경험에 의존하던 기존의 의사결정 방식을 데이터 기반으로 전환하는 중요한 계기를 만들었습니다.

Learnings (배운 점)

이해관계자(정책 담당자)가 모델의 결과를 쉽게 이해하고 신뢰할 수 있도록, '왜' 이런 결과가 나왔는지 설명하는 능력(Explainability)의 중요성을 깨달았습니다. 복잡한 모델이라도 쉬운 시각화와 명확한 설명이 동반되어야 실질적인 영향력으로 이어진다는 것을 배웠습니다.
